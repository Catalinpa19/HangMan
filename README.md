# HangMan Game (WPF, MVVM)

A desktop HangMan game built with WPF using the MVVM architectural pattern.

## ✨ Features
- User login and registration
- Classic HangMan gameplay
- Multiple word categories
- Save and load games
- Player statistics tracking
- Keyboard input support
- Local JSON data storage

## 🛠 Technologies
- C#
- WPF
- MVVM pattern
- JSON Serialization

## 📁 Data Storage
- Users data
- Saved games
- Statistics

## 🚀 Future Improvements
- Difficulty levels
- Better UI animations
- Leaderboard system
- Sound effects
